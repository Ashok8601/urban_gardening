‚ùì Top Interview Questions

Q1. What is the difference between a list and a tuple?

**A.** The key difference is **mutability**. Lists are mutable (can be changed), while tuples are immutable (cannot be changed). Lists use square brackets `[]`, tuples use parentheses `()`. Tuples are faster and safer for fixed data.

Q2. What is a "docstring" in Python?

**A.** It's a string literal that occurs as the first statement in a module, function, class, or method definition. It's used to **document the code** and can be accessed using the `.__doc__` attribute. 

def my_func():
    """This is the docstring for my_func."""
    pass

Q3. Explain Python's Global Interpreter Lock (GIL).

**A.** The GIL is a **mutex** (lock) that prevents multiple native threads from executing Python bytecodes *simultaneously* in a single process. It makes Python effectively **single-threaded** for CPU-bound tasks, even on multi-core machines. It exists to simplify memory management and C-extension integration.
